EBS Merchant Gateway For eShop

== Description ==

This plugin is  a new add-on plugin for eShop WordPress plugin which acts as an additional gateway along with other gateways in eShop plugin. This plugin is very useful for INDIAN merchant's who are having EBS merchant accounts.

Features:-
* This plugin uses the HTTPS SSL secured payment system for giving more secure to all users while purchasing any product from eShop powered store.
* Displays an awesome payment form which is provided by EBS payment solutions to all users who are in the process of buying products by selecting EBS as payment system.

== Installation ==
1. Download the Plugin by searching in WP Plugins repository.
2. Upload the "ebs-gateway-eshop" folder to your site's 'wp-content/plugins' directory.
3. Activate the plugin using Activate button.
4. Proceed for ESHOP settings page & proceed further by filling details about your EBS merchant account & get benefits for your eShop store(s) products.
